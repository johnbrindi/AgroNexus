# Mobile App - AgroNexus (Expo)

## Setup & Running
1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Run the app:**
    ```bash
    npx expo start
    ```

## Folder Structure
- `App.js`: Main entry point and navigation.
- `src/components`: Reusable UI components (`BottomNav`, etc.).
- `src/screens`: Feature screens (`Auth`, `Dashboard`, `Devices`, etc.).
- `src/styles`: Theme and global styles.
